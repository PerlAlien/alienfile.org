NAME

Alien::Build::Plugin::Core - Core Alien::Build plugins

VERSION

version 2.68

SYNOPSIS

 use alienfile;
 # core plugins are already loaded

DESCRIPTION

Core plugins are special plugins that are always loaded, usually first.

Alien::Build::Plugin::Core::CleanInstall
Alien::Build::Plugin::Core::Download

This contains the default machinery for downloading packages, if no other download plugin or commands are provided.

Alien::Build::Plugin::Core::FFI
Alien::Build::Plugin::Core::Gather
Alien::Build::Plugin::Core::Legacy

Add interoperability with Alien::Base::ModuleBuild

Alien::Build::Plugin::Core::Override

The machinery which allows you to override the type of install with the ALIEN_INSTALL_TYPE environment variable.

Alien::Build::Plugin::Core::Setup
Alien::Build::Plugin::Core::Tail

SEE ALSO

Alien::Build, Alien::Build::Plugin
