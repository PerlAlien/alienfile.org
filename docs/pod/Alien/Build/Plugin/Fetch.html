NAME

Alien::Build::Plugin::Fetch - Fetch Alien::Build plugins

VERSION

version 2.59

SYNOPSIS

 use alienfile;
 share {
   start_url 'http://ftp.gnu.org/gnu/make';
   plugin 'Download';
 };

DESCRIPTION

Fetch plugins retrieve single resources from the internet. The difference between a Fetch plugin and a Download plugin is that Download plugin may fetch several resources from the internet (usually using a Fetch plugin), before finding the final archive. Normally you will not need to use Fetch plugins directly but should instead use the Alien::Build::Plugin::Download::Negotiate plugin, which will pick the best plugins for your given URL.
