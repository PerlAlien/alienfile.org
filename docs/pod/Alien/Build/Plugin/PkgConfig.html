NAME

Alien::Build::Plugin::PkgConfig - PkgConfig Alien::Build plugins

VERSION

version 2.77

SYNOPSIS

 use alienfile;
 plugin 'PkgConfig' => (
   pkg_name => 'foo',
 );

DESCRIPTION

PkgConfig plugins use pkg-config or a compatible library to retrieve flags at probe and gather stages.

Alien::Build::Plugin::PkgConfig::CommandLine

Use the command-line pkg-config or pkgconf to get compiler and linker flags.

Alien::Build::Plugin::PkgConfig::LibPkgConf

Use the XS PkgConfig::LibPkgConf to get compiler and linker flags.

Alien::Build::Plugin::PkgConfig::MakeStatic

Convert .pc file to use static linkage by default.

Alien::Build::Plugin::PkgConfig::MakeStatic

Choose the best plugin to do pkg-config work. The best choice is typically platform and configuration dependent.

Alien::Build::Plugin::PkgConfig::PP

Use the pure-perl PkgConfig to get compiler and linker flags.
