NAME

Alien::Build::Plugin::Probe - Probe Alien::Build plugins

VERSION

version 2.73

SYNOPSIS

look for libraries in known location:

 use alienfile;
 plugin 'Probe::CBuilder' => (
   cflags => '-I/opt/libfoo/include',
   libs   => '-L/opt/libfoo/lib -lfoo',
 );

look for tools in the path:

 use alienfile;
 plugin 'Probe::CommandLine' => (
   command => 'gzip',
   args    => [ '--version' ],
   match   => qr/gzip/,
   version => qr/gzip ([0-9\.]+)/,
 );

Use vcpkg for Visual C++ Perl:

 use alienfile;
 plugin 'Probe::Vcpkg' => 'libffi';

DESCRIPTION

Probe plugins try to find existing libraries and tools already installed on the system. If found they can be used instead of downloading the source from the internet and building.

Alien::Build::Plugin::Probe::CBuilder

Use ExtUtils::CBuilder to probe for existing installed library.

Alien::Build::Plugin::Probe::CommandLine

Execute commands to probe for existing tools.

Alien::Build::Plugin::Probe::Vcpkg

Use Win32::Vcpkg to probe for existing installed library.
