NAME

Alien::Build::Log::Default - Default Alien::Build log class

VERSION

version 2.59

SYNOPSIS

 Alien::Build->log("message1");
 $build->log("message2");

DESCRIPTION

This is the default log class for Alien::Build. It does the sensible thing of sending the message to stdout, along with the class that made the log call. For more details about logging with Alien::Build, see Alien::Build::Log.

METHODS

log

 $log->log(%opts);

Send single log line to stdout.
