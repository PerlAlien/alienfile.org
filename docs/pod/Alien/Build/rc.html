NAME

Alien::Build::rc - Alien::Build local config

VERSION

version 2.68

SYNOPSIS

in your ~/.alienbuild/rc.pl:

 preload 'Foo::Bar';
 postload 'Baz::Frooble';

DESCRIPTION

Alien::Build will load your ~/.alienbuild/rc.pl file, if it exists before running the alienfile recipe. This allows you to alter the behavior of Alien::Build based Aliens if you have local configuration requirements. For example you can prompt before downloading remote content or fetch from a local mirror.
