NAME

Test::Alien::Diag - Print out standard diagnostic for Aliens in the test step.

VERSION

version 2.68

SYNOPSIS

 use Test2::V0;
 use Test::Alien::Diag qw( alien_diag );

DESCRIPTION

This module provides an alien_diag method that prints out diagnostics useful for cpantesters and other bug reports that gives a quick summary of the important settings like clfags and libs.

FUNCTIONS

alien_diag

 alien_diag @aliens;

prints out diagnostics for each given alien. Each alien must be the class name of an alien.

[version 2.68]

 alien_diag @aliens, \%options;

Starting with Alien::Build 2.68, you can provide an option hash to adjust the behavior of alien_diag. Valid options are:

properties

Additional properties to display in the diagnostic. Useful when you have an Alien with custom properties defined in the subclass.

list_properties

Additional properties that are returned as a list to display in the diagnostic. Useful when you have an Alien with customer properties that return a list.
